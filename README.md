# Master Sis - Sistema de Gestão para Academias

Um sistema focado na gestão de alunos, modalidades, graduações, planos e faturamento para academias de artes marciais.

## 🚀 Tecnologias Utilizadas
- Java 17
- Spring Boot 3
- Spring Data JPA
- PostgreSQL
- Flyway (para versionamento de banco de dados)
- Lombok
- Spring Validation
